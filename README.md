# CS111: Fundamentals of Programming

นี่คือเทมเพลต GitHub Codespaces สำหรับวิชา **CS111: Fundamentals of Programming** ซึ่งเราได้เตรียมสภาพแวดล้อมสำหรับการเขียน Python 3.12 ไว้ให้พร้อมใช้งานทันทีบนเบราว์เซอร์

## เริ่มต้นใช้งานอย่างรวดเร็ว (Quick Start)

เพื่อให้งานของเราถูกสำรองไว้บน GitHub ส่วนตัวอย่างปลอดภัย ให้ทำตามขั้นตอนดังนี้:

1.  **เปิด Terminal** (กด `` Ctrl + ` ``) แล้วพิมพ์คำสั่ง:
    ```bash
    ./.devcontainer/setup.sh
    ```
2.  **ยืนยันตัวตนกับ GitHub**: ทำตามขั้นตอนที่ปรากฏใน Terminal:
    - เลือก **Yes** เพื่อสร้าง SSH key ใหม่
    - กด **Enter** ข้ามขั้นตอนรหัสผ่าน (Passphrase) และชื่อไฟล์
    - **คัดลอกรหัส 8 หลัก** ที่เห็นใน Terminal
    - กด **Enter** เพื่อเปิดหน้าเว็บ แล้ววางรหัสเพื่อยืนยันตัวตน (Authorize)
    - เมื่อเสร็จแล้ว ระบบจะสร้าง repository ชื่อ `26cs111python` ในบัญชีของเราให้อัตโนมัติ

## การทำงานของเรา

-   **โฟลเดอร์ `homework/`**: ให้เราเขียนโค้ดและทำงานทั้งหมดไว้ในโฟลเดอร์นี้ (แบ่งตามสัปดาห์ w01, w02, ...)
-   **ระบบสำรองงานอัตโนมัติ (Autosync)**: ไม่ต้องกังวลเรื่องลืมเซฟงาน ระบบจะแอบทำการ Backup (Commit & Push) งานในโฟลเดอร์ `homework/` ขึ้น GitHub ให้เราทุกๆ 10-30 นาที หากเราไม่ได้พิมพ์อะไรนานๆ
-   **เครื่องมือพร้อมใช้**: เราติดตั้ง Python 3.12, ตัวตรวจไวยากรณ์ (mypy), และ Extension ที่จำเป็นไว้ให้แล้ว
